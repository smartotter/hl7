Title: Mammography - Partial View Option for Modalities

References:
 RAD TF-2:4.8.4.1.2.3

Only Modalities which support the Partial View option are required
to run test 3915.

You will be asked to C-STORE an image which is part of a mosaic
used to cover an area of a breast that is larger than the detector.
